# NXlive — 넥슨 게임 라이브 스트리밍 플랫폼 (PRD)

## 1. 개요
넥슨 게임 유저가 라이브 방송을 **발견하고 시청**하는 웹 스트리밍 플랫폼.
Mock 데이터 기반 MVP. 실제 스트리밍 서버·인증 없이 프론트엔드 중심으로 UX/흐름을 검증한다.

## 2. 목표 (MVP)
- 유저가 라이브 방송을 쉽게 발견한다 (홈/검색).
- 방송을 시청하며 채팅·정보를 함께 본다 (라이브 시청).
- 스트리머 채널을 탐색한다 (채널).
- 하이라이트 클립을 소비한다 (클립).

## 3. 대상 사용자
넥슨 게임을 즐기는 외부 유저. **인증 없음** (로그인/회원 기능 제외, 익명 열람).

## 4. 범위 (5개 페이지)
| # | 페이지 | 핵심 목적 |
|---|---|---|
| 1 | 홈 | 라이브 방송 발견 — 추천/카테고리/실시간 인기 |
| 2 | 라이브 시청 | 플레이어 + 실시간 채팅 + 방송 정보 |
| 3 | 채널 | 스트리머 프로필 + 지난 방송/클립 |
| 4 | 클립 | 하이라이트 클립 피드 |
| 5 | 검색 | 게임/스트리머/클립 통합 검색 |

## 5. 비범위 (Out of Scope)
- 실제 스트리밍/트랜스코딩 인프라 (Mock 영상/썸네일 대체)
- 인증·회원·결제·후원
- 방송 송출(스트리머 스튜디오)
- 백엔드 API (Mock 데이터 모듈로 대체)

## 6. 데이터 (Mock)
- 게임, 스트리머(채널), 라이브 방송, 클립, 채팅 메시지 → 정적 mock 모듈로 제공.

## 7. 인상 키워드
역동성 · 혁신/모던함 (게임 라이브 스트리밍 도메인)

## 8. 성공 기준
- 5개 페이지 라우팅 및 상호 이동이 자연스럽다.
- 발견 → 시청 핵심 플로우가 3클릭 이내로 완결된다.
- 반응형(데스크톱 우선, 태블릿 대응).
